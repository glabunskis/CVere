\section*{Key Projects}

\textbf{Connector Administration Platform}\\
Built backend functionality in Spring Boot for a centralized connector tracking platform used across ABN Amro service teams, adding services, APIs, DTOs, and exception handling for external integrations. Led front-end development in Vue.js, delivering features, views, bug fixes, and Vitest-based tests to reduce data duplication and improve cross-team visibility.\\[4pt]
\textbf{Distributed Food Chain Management System}\\
Built a distributed restaurant and order management system of four microservices using Spring and Hibernate, achieving over 90\% test coverage with JUnit and Mockito and validating endpoints with Postman.\\[4pt]
\textbf{Live Q\&A Web Application}\\
Built an interactive Q\&A web application using Spring Boot, Hibernate, and REST APIs, contributing requirement analysis, Scrum planning, and front-end coordination during feature delivery.
